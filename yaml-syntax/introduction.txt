INTRODUCTION TO YAML
====================
* YAML is a light-weight, human-readable, data-serialization language.It is primarily designed to make 
  the format easy to read while including advanced features.
* YAML stands for "YAML Ain't Markup Language" 

XML vs JSON vs YAML
===================
YAML is a superset of JSON, which means all the features of JSON can be found in YAML. for example if you
compare javascript with typescript, typescript will be YAML and javascript will be JSON
